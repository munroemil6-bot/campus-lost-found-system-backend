# Campus Lost & Found System

This repository contains a Python FastAPI backend and a React frontend scaffold for a campus lost-and-found application.

- `backend/` contains the FastAPI application, database setup, models, schemas, routers, and utilities.
- `frontend/` contains the React application built with Vite, routes, components, and API service helper.
- `docker-compose.yml` orchestrates the frontend and backend services.
- `Dockerfile.backend` and `Dockerfile.frontend` define separate container images.
