This root folder now includes a landing page for the Dockerized app. Start the services, then use the links below to open the frontend or backend API.

Frontend app — http://localhost:3000 Backend API docs — http://localhost:8000/docs Backend root API — http://localhost:8000

Run the app from the project root using Docker Compose:

docker compose up --build
Note: The links work when the backend and frontend containers are running. If the frontend does not load, make sure the Docker app has started and you are using http://localhost:3000.
