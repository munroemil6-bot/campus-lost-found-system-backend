Campus Lost & Found

This repository is configured for public deployment using GitHub Actions and an external hosting provider like Render.

The frontend and backend are built separately as Docker images, and the frontend is linked to the backend by the `FRONTEND_API_URL` secret.

Once deployed, your live site will be available at the Render frontend service URL.

If you need the app locally, start it with:

docker compose up --build -d

Then open the app on your local machine or LAN using your host IP.
